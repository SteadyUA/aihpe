# Идеи по улучшению UI/UX сессий

## 1. Визуальная группировка сессий (Tree View)
Логичнее отображать клонированные сессии (варианты) как дочерние элементы исходной сессии, а не плоским списком.

**Реализация:**
*   Добавить поле `parentId` в структуру `Session`.
*   В `App.tsx` собирать/рендерить дерево сессий.
*   Добавить возможность сворачивать/разворачивать ветки (родительские сессии), чтобы скрывать промежуточные или неудачные варианты.

## 2. Умная индикация статуса в сайдбаре
Улучшение информативности списка сессий, особенно при фоновой работе.

**Реализация:**
*   **Зеленая галочка (Fade out):** При завершении генерации на табе сессии на несколько секунд появляется зеленый индикатор успеха.
*   **Красный индикатор:** Явный значок ошибки, если фоновая генерация упала.
*   **Пульсация:** Мягкая анимация рамки или фона таба, пока в этой сессии идет генерация (статус `busy/generating`).

## 3. Группировка по времени (Timeline)
Альтернатива или дополнение к дереву — визуальное разделение сессий по времени создания/обновления.

**Реализация:**
*   Использовать поле `updatedAt` для сортировки.
*   Вставлять в `SessionBar` разделители (заголовки): "Сегодня", "Вчера", "На прошлой неделе".
